The plugin sends a message to the HelloTXT social notification network when a post is published in WordPress.

== Description ==

This plugin is designed to send notifications to the helloTXT.com social 
network notification service when a new post is published in WordPress.



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder 'hellotxt' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the HelloTXT settings page to configure plugin.



== Frequently Asked Questions ==

= Do I need an accounts on HelloTXT.com? =

Yes, an accounts on HelloTXT.com is required for this plugin to function.
